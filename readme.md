Screentime is a chrome extension that helps you track and limit your time on distracting websites. With Screentime, you can set daily limits for specific websites, receive notifications when you're approaching your limit, and view detailed reports of your browsing activity.

## Features
1. Set daily time limits for specific websites.
2. Receive notifications when you're nearing your time limit.
3. View detailed reports of your browsing activity.
4. Customize settings for different days of the week.
5. Easy-to-use interface.
6. Keep your data locally on your device.

## Installation

### For Users (from Chrome Web Store)
1. Download the Screentime extension from the Chrome Web Store.
2. Click "Add to Chrome" and confirm the installation.
3. Once installed, click on the Screentime icon in the toolbar to open the extension
4. Configure your settings and start tracking your screen time!

### For Developers (local installation)
1. Clone this repository or download the source code
2. Generate PNG icons from the SVG source:
   ```bash
   ./generate-icons.sh
   ```
   Or follow the instructions in `icons/README.md`
3. Open Chrome and navigate to `chrome://extensions/`
4. Enable "Developer mode" (toggle in top-right corner)
5. Click "Load unpacked" and select the `screentime` directory
6. The extension is now installed and ready to use!

For detailed installation and troubleshooting instructions, see [INSTALL.md](INSTALL.md)

## Usage
1. Open the Screentime extension by clicking on its icon in the Chrome toolbar.
2. Navigate to the "Settings" tab to set your daily time limits for specific websites.
3. Use the "Reports" tab to view your browsing activity and see how much time you've spent on each site.
4. Adjust your settings as needed to better manage your screen time.
